One-time setup required

  1. Go to Google Cloud Console → APIs & Services → Credentials
  2. Create an OAuth 2.0 Client ID (Web application type)
  3. Add your GitHub Pages URL to Authorised JavaScript origins
  4. Enable Google Search Console API in your project
  5. Paste your Client ID below and sign in

Your data stays in your browser. No data is sent to any third-party server. Requires GSC Owner or Full User access on each property.
